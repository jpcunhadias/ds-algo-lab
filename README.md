# DSA Learning Platform

A Python-based educational platform for studying data structures and algorithms with visual implementations, testing capabilities, and future exercise support.

## Features

- **Modular Design**: Each data structure and algorithm is self-contained with its own visualization and tests
- **Visual Implementations**: Step-by-step, interactive, and animated visualizations
- **Comprehensive Testing**: Unit tests and visual tests for all components
- **Educational Focus**: Well-commented code with explanations

## Project Structure

```
ds-algo-lab/
├── src/
│   ├── data_structures/     # Core DS implementations
│   ├── algorithms/          # Algorithm implementations
│   ├── visualization/       # Visualization engine
│   ├── testing/             # Testing framework
│   └── utils/               # Utilities
├── examples/                # Example usage scripts
├── tests/                   # Test files
└── requirements.txt         # Python dependencies
```

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

See the `examples/` directory for usage examples of each component.

## Running Tests

```bash
pytest tests/
```
